## 4. Final `mathlib.py`

```python

def triangle_number(n: int) -> int:
    """Return the nth triangular number."""
    if n < 0:
        raise ValueError("n must be non-negative")
    return n * (n + 1) // 2



def rectangular_number(n: int, m: int) -> int:
    """Return m times the nth triangular number."""
    return triangle_number(n) * m
```
